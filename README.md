## 프로젝트 개요

![summary](https://github.com/UruruLab/Ururu-Backend/blob/main/readme/landing-res/2.png?raw=true)

**Ururu**는 AI 기술을 활용한 뷰티 커머스 플랫폼입니다. 사용자 맞춤형 상품 추천, 공동구매 시스템, 결제 처리 등 복잡한 이커머스 비즈니스 로직을 안정적으로 처리하는 서비스를 제공합니다.
